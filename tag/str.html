Python学习笔记-字符串操作

 2013-09-22 by particle128

注意：tr支持的操作，基本上unicode也支持。此外unicode还多两个函数unicode.isnumeric() unicode.isdecimal()

（一） 具备的操作

str作为序列元素，包含其他序列共同的操作：
1. “hello” [not] in "hello world"
2. "hello" + " world"
3. 5"hello" 或 “hello”5
4. "hello"[3] "hello"[1:2] "hello"[1:2:2]
5. len("hello") min("hello") max("hello")
6. "hello".index('o') "hello".count('l')

（二） 不具备的操作

str同tuple一样，属于不可变序列。可变序列包括list和bytearray。因此str不具备可变序列的操作：
1. s[i] = x s[i:j] = t 【t是任何可迭代对象】 s[i:j:k] = t 【t需要和s[i:j:k]有相同数量的元素】
2. del s[i:j] del s[i:j:k] s.remove(x) => del s[x]
3. s.insert(i, x) s.append(x) s.extend(t) 【t是任何可迭代对象】
4. s.index(x[, i[, j]]) 【匹配x的最小索引，可以给定一个范围i<=idx5. .reverse() s.sort([cmp[, key[, reverse]]])
6. s.pop([i]) => x = s[i]; del s[i]; return x

（三） 字符串方法

str.capitalize()第一个字母大写 str.title() 每一个单词首字母大写
str.count(sub[, start[, end]])
str.endswith(suffix[, start[, end]])返回True或False 同理str.startswith
str.find(sub[, start[, end]]) 找不到返回-1 同理str.rfind
str.index(sub[, start[, end]])找不到raise ValueError 同理str.rindex
str.format(args, *kwargs) 同格式化字符串
islower isupper isspace isdigit isalpha isalnum
str.ljust(width[, fillchar])左对齐，可以选择填充字节，默认空格。 同理str.rjust，str.zfill(width)相当于str.rjust(width,'0')
str.lower() str.upper() str.swapcase()交换大小写
str.partition(sep) 划分 'what the fuck'.partition('the') => ('what ','the',' fuck') 同理rpartition
str.join(iterable) 把可迭代对象（比如序列）拼接成字符串，用str隔开每一个可迭代对象中的元素
str.translate(string.maketrans(from,to) [,deletechars]) 要么进行字母的替换，要么进行字母的删除（第一个参数为None），要么同时进行
str.strip([chars]) 默认去掉whitespace 同理str.lstrip str.rstrip
str.split([sep]) 默认whitespace作为分隔符

（四） string方法

string.maketrans(from,to) 返回translation table（from中的字符被to中同一位置的字符替换），from和to长度要相同
string.capwords(s[,sep]) 比titile更加准确的首字母大写函数。先split([sep])，再capitalize()，再join()

（五） 格式化字符串

"%(language)s has %(number)03d quote types."  %  {"language": "Python", "number": 2}

（六）字符串常量

string.digits
string.hexdigits
string.octdigits
string.lowercase
string.uppercase
string.whitespace
string.punctuation
string.printable
